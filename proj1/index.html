Project 1: Images of the Russian Empire

Overview

This project uses computer vision techniques to colorize early Prokudin-Gorskii glass plate photographs. The photographs were originally captured as three grayscale images (R, G, B channels) stacked vertically. I separated and aligned them to reconstruct full-color images. In the process, I learned about image pyramid techniques and saw how three monochrome images can be combined into striking color photos.

Part 1: Naive Approach — Iterative

My first approach was to convert each image into an array and try exhaustive search over pixel shifts (x and y directions) to find alignment. I tested both L2 norm and normalized cross-correlation (NCC); for smaller images, NCC produced more stable results. To improve alignment, I cropped 10% of the image borders and compared results. Cropping both reference and target before shifting gave the best alignment.
